# Video Conferencing Web App (Django)

## 📌 Overview
This project is a **real-time video conferencing web application** built using **Python (Django framework)**.  
It integrates APIs and libraries to enable video calls, audio calls, and chat features, with a secure and user-friendly interface.

## ⚙️ Features
- 🎥 **Video Conferencing**: Peer-to-peer video calls using WebRTC.
- 🎙️ **Audio Calls**: Audio-only sessions for low bandwidth.
- 💬 **Chat Integration**: Real-time messaging alongside video.
- 👥 **Multi-user Support**: Group meetings and collaboration.
- 🔒 **Secure Connections**: Encrypted communication channels.
- 🖥️ **Dashboard**: Manage meetings and participants easily.

## 🛠️ Tech Stack
- **Backend**: Django
- **Libraries**: `asgiref`, `sqlparse`, `tzdata`
- **Frontend**: HTML, CSS, JavaScript
- **Real-time Communication**: WebRTC, Django Channels
- **Database**: SQLite.
